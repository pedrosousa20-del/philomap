## 1. O que eu pude compreender?

Ao terminar a visualição do conteúdo disponibilizado pelo Porfessor Iury Gomes a respeito dos conceitos básicos do **CSS** pude compreender que é uma linguagem utilizada para definir totalmente o design da aparência e estilo de uma página web como: cores, fontes, espaçamentos, layouts. Sem o CSS, o site seria apenas uma estrutura de texto e elementos básicos do HTML. Sem em uma boa visualização.

## 2. Por que usar CSS e Arquivos Externos?


O uso de um **arquivo externo** (como o `style.css`) é o mais recomendado por três motivos principais:
* **Organização:** Mantém o código HTML limpo, focado apenas na estrutura.
* **Manutenção:** Facilita a alteração do estilo de várias páginas ao mesmo tempo, editando apenas um arquivo.
* **Performance:** O navegador carrega o estilo uma vez e pode reutilizá-lo em outras páginas do mesmo site.

## 3. Glossário de Propriedades CSS

Aqui estão as principais propriedades utilizadas para estilizar os elementos:

* **color:** Define a cor do texto de um elemento. [00:23:25]
* **background-color:** Define a cor de fundo de um elemento ou seção. [00:22:28]
* **margin:** Controla o espaço **fora** da borda do elemento, criando distância entre ele e outros componentes. [00:14:10]
* **padding:** Controla o espaço **dentro** da borda do elemento, ou seja, o respiro entre o conteúdo e a sua borda. [00:18:28]
* **display: flex:** Ativa o Flexbox, uma ferramenta poderosa para alinhar e distribuir elementos em colunas ou linhas de forma organizada. [00:32:22]
* **border-radius:** Arredonda as quinas dos elementos (como botões e imagens). [00:43:05]
* **font-family:** Define qual tipo de letra (fonte) será exibida no site. [00:25:48]

## 4. O Conceito de Classes

As **classes** são identificadores que adicionamos às tags HTML (ex: `<p class="texto-destaque">`) para que possamos selecionar elementos específicos no CSS. 

Diferente de selecionar apenas a tag `p` (que mudaria todos os parágrafos do site), as classes nos dão controle total para estilizar apenas os itens que desejamos, permitindo criar estilos personalizados para diferentes partes da mesma página. No CSS, as classes são identificadas por um ponto antes do nome (ex: `.texto-destaque`). [01:04:15]
